# WorkLog Assistant Icon Files

此目录应包含应用图标文件：
- icon.png (用于Linux/Mac)
- icon.ico (用于Windows)

## 当前状态

目前只有README占位文件，需要添加实际图标。

## 如何创建图标

### 方法一：使用在线工具
1. 访问 https://www.favicon-generator.org/
2. 上传您的图标图片
3. 下载生成的icon.png和icon.ico

### 方法二：使用图像编辑软件
1. 使用Photoshop、GIMP等软件
2. 创建512x512像素的图标
3. 导出为PNG格式
4. 使用在线工具转换为ICO格式

### 方法三：使用Electron默认图标
暂时不添加图标，Electron会使用默认图标。

## 推荐图标设计

- 尺寸：512x512像素
- 格式：PNG (透明背景)
- 颜色：蓝色或绿色主题
- 内容：字母"WL"或工作相关图标
