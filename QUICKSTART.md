# WorkLog Assistant - 快速开始指南

## 当前版本

**v0.3.0**

## 首次使用

### 1. 启动应用

**开发者：**
```bash
npm run electron:dev
```

### 2. 配置 API 密钥

1. 点击右上角 ⚙️ 进入设置
2. 在"API 设置"中填写 API Key、Base URL、模型名称
3. 配置会自动保存

### 3. 开始监控

应用启动后自动开始记录，无需手动操作。点击顶部按钮可暂停/恢复。

### 4. 生成摘要

- 点击"摘要"标签页 → "生成摘要"按钮（基于回溯时间）
- 或选择开始/结束时间 → "按时间段生成"（自选范围）

## 配置选项

### 记录设置
- **记录间隔**: 每次检查间隔（秒），最小 10 秒，建议 60 秒

### 摘要设置
- **回溯时间**: 摘要包含的活动时间范围（分钟）
- **最大令牌数**: AI 生成的最大 token 数
- **压缩活动**: 合并相同应用的活动记录
- **AI 提示词**: 自定义提示词（留空使用默认）
- **总结周期**: 配置多个时间段，到点自动生成摘要

### 费用预估
- 输入/输出价格（元/百万 token）
- 每日工作时长
- 自动计算每日/每月预估费用

### 通用设置
- 开机自启动、最小化浮窗、主题选择（5 种）

## 常见问题

**应用无法启动？** 检查 Node.js 版本（建议 18+），重新 `npm install`

**AI 摘要失败？** 检查 API 密钥、网络连接、API 端点配置

**如何退出？** 右键托盘图标 → 退出（关闭窗口只会最小化到托盘）
