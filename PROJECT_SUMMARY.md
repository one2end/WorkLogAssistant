# WorkLog Assistant JavaScript版本 - 项目总结

## 项目概述

基于 Electron + Vue 3 的工作日志助手，自动监控工作活动并生成 AI 摘要。当前版本 **v0.3.0**。

## 技术栈

- **Electron**: 跨平台桌面应用框架
- **Vue 3**: 前端框架
- **Vite**: 构建工具
- **Axios**: HTTP 客户端
- **Marked**: Markdown 解析

## 已实现功能

### 核心功能

1. **窗口监控** (`windowMonitor.js`) - 跨平台活动窗口监控，智能去重
2. **AI 摘要** (`summarizer.js`) - 支持自定义 API、压缩活动、自定义提示词、时间段摘要、回溯摘要
3. **自定义总结周期** - 配置多个时间段，到点自动生成该时段摘要
4. **费用预估** - 根据配置参数实时计算 API 调用费用
5. **系统托盘** (`main.js`) - 最小化到托盘，右键菜单，双击显示
6. **浮窗模式** (`float.html`) - 悬浮窗显示当前应用和持续时间，可拖拽
7. **数据存储** (`storageManager.js`) - JSON 文件存储，支持时间范围查询
8. **配置管理** (`configManager.js`) - 深度合并配置，修改即时生效
9. **日志管理** (`logManager.js`) - 实时日志记录和文件存储

### 用户界面

1. **主界面** (`App.vue`) - Vue 3 组件，5 种主题，响应式布局
2. **统计面板** - 活动数量、应用数量、记录时长、未操作电脑时长
3. **图表展示** - 应用使用时长排名、子活动次数排名
4. **自选时间段摘要** - datetime-local 选择器，按时间范围生成摘要

### 系统集成

- 开机自启动、自动更新检查、数据导出、单实例运行、托盘快捷操作

```

│   ├── storageManager.js # 数据存储
│   ├── logManager.js     # 日志管理
│   └── float.html        # 浮窗页面
├── src/                   # Vue 前端
│   ├── App.vue           # 主组件
│   └── main.js           # 入口文件
├── build/                 # 构建资源（图标等）
├── package.json
├── vite.config.js
└── README.md
```
